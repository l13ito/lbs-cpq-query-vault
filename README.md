# lbs-cpq-query-vault

A curated vault of SQL and analytical queries for the LBS Configure, Price, Quote (CPQ) platform.

## Overview

This repository serves as a centralized library of reusable queries used to support CPQ operations, reporting, and data analysis. It provides a single source of truth for query logic across teams working with the CPQ system.

## Contents

- **Pricing Queries** — Queries related to pricing rules, discount calculations, and price book lookups.
- **Configuration Queries** — Queries for product configurations, option sets, and constraint validations.
- **Quote Queries** — Queries for quote lifecycle management, approvals, and pipeline reporting.
- **Analytics** — Aggregated and summary queries for dashboards and business intelligence.

## Usage

Browse the relevant folder for the query you need. Each query file includes a header comment describing:

- **Purpose** — What the query does.
- **Parameters** — Any input variables or filters.
- **Dependencies** — Required tables, views, or objects.
